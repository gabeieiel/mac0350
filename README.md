# mac0350
Repositório da disciplina MAC0350 - Introdução ao Desenvolvimento de Software

Meu projeto individual é um repositório de Sinuca. A ideia é que hajam duas páginas, **Atletas** e **Modalidades**, onde o usuário poderá adicionar, buscar, editar e deletar instâncias de ambos os casos em cards.

Os arquivos estavam organizados da seguinte forma:

PROJETO INDIVIDUAL
|_ main.py
|_ static
|  |_  styles.css
|_ templates
   |_  index.html
   |_  atletas.html
   |_  fragmentos
       |_ card_atleta.html
       |_ card_modalidade.html
       |_ form_atleta.html
       |_ form_modalidade.html
       |_ lista_atletas.html
       |_ lista_modalidades.html
